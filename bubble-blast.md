## How The Advantage Works

Bubble Blast / Spells 'N Whistles / Trials of Atlantis features **persistent bubbles** with credit values:

**Mechanic:**
- Bubbles with credit values appear on reels
- Each spin → Bubbles move **up one row**
- Bubble lands on "catcher" symbol → Win credit value
- Bubbles can be triggered **multiple times** (don't disappear)

---

## PLAY WHEN

⚠️ **CRITICAL:** Do NOT count bubbles on top row—they leave next spin.

<strong>Primary Requirement:</strong>
- Total bubble credit value ≥ <strong>5x</strong> bet size
- Example: $5 bet → Need $25+ in bubble values

**Also Check:**
- Bubbles below bottom row (moving into play next spin)
- If credit value obscured → Play anyway to reveal it

**Bubble Position Check:**
| Row | Counts Toward Total? |
|-----|---------------------|
| Top row | ❌ No (exits next spin) |
| Middle rows | ✅ Yes |
| Bottom row | ✅ Yes |
| Below bottom | ✅ Yes (incoming) |

---

## DO NOT PLAY WHEN

- Total bubble value below <strong>5x</strong> bet
- Only bubbles on top row
- No bubbles visible

---

## STOP WHEN

- Bubbles collected and remaining total drops below <strong>5x</strong> bet
- All bubbles move off top of screen

---

## COMMON MISTAKES

- Counting top row bubbles (they're gone next spin)
- Not checking below bottom row for incoming bubbles
- Not checking all denominations

---

## Additional Notes

**Bubble Patterns:**
- Tend to appear in big spurts
- Game goes cold between spurts
- Can appear below bottom row

**Bubble Behavior:**
- Don't go away after triggering
- Can win same bubble **multiple times**

**Mini/Maxi Jackpots:**
- Can appear inside bubbles

**Spin Button Note:**
- After big win, spin button immediately spins (doesn't skip win animation)

**Denominations:**
- Multiple denominations available
- Check all bet levels for plays
